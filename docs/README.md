- **Frontend**: React 18, TypeScript, Vite, TailwindCSS, shadcn/ui
- **Authentication**: ASP.NET Core Identity, JWT, OAuth (GitHub, Discord)
- **Email**: FluentEmail with Liquid templates

## What is Crystal?

Crystal is a modern, production-ready template for building full-stack web applications with .NET and React. It includes:

- Complete authentication system (Email/Password + OAuth)
- Modern React UI with dark/light theme support
- Email confirmation and password reset flows
- Type-safe client library for API communication
- Comprehensive error handling and logging
- Docker support for deployment
